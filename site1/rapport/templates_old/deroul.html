Déroulement du Stage

Observation des Métiers du Pôle Technique

Durant mon stage, j’ai pu observer différents métiers dans le domaine de l’informatique. Au début de mon stage, j’ai eu l’occasion de découvrir des métiers du pôle technique avec Julien Deludet, technicien qui administre et gère des réseaux et Zinedine Khazerouni, un ingénieur en apprentissage.

Le premier jour, j’ai pu observer Julien utiliser différents outils pour travailler comme un VNC (Microsoft TeamViewer), qui est également un VPN SSL servant à relier son ordinateur à un autre ordinateur. Il m’a également montré et expliqué ce qu’est un GLPI (Gestionnaire Libre de Parc Informatique) qui sert à afficher le nom des machines du domaine informatique que l’entreprise gère. Le GLPI permet d’afficher l’utilisation de la mémoire vive, de la mémoire de la machine, et la charge du processeur du serveur. C’est grâce à ces outils que Julien a pu créer des sauvegardes de bases de données sur des serveurs distants et optimiser le stockage en supprimant des anciennes sauvegardes inutiles.

Il m’a aussi expliqué l’utilité d’un pare-feu qu’il configurait pour un client. Ce dispositif sert à filtrer le trafic réseau et à bloquer les accès non autorisés. Par la suite, il m’a également montré comment les machines échangent des informations entre elles. Elles utilisent des paquets qui partent d’un ordinateur, sont expédiés jusqu’au routeur qui assure la connexion à internet, et arrivent à un autre ordinateur auquel ils peuvent accéder via un port.

Par analogie, une machine serait une maison qui pour communiquer avec la maison d’un autre lotissement, utiliserait une voiture (paquet) qui arriverait à un rond-point (routeur). Le rond-point redirigerait la voiture vers le bon lotissement. Celle-ci entrerait dans la maison de destination en passant par le garage (port ouvert) et délivrerait son message (lettre, colis…).

Installation d'un Serveur Linux

Plus tard, avec Zinedine, j’ai eu l’occasion d’installer un serveur Linux (Debian, un système d’exploitation open-source) sur une machine virtuelle. Une machine virtuelle est un environnement virtualisé qui fonctionne sur un ordinateur.

Zinedine m’a proposé d’installer Linux de manière non graphique, comme ils le font pour les prestations clients. Debian est un équivalent open-source de Windows, ce qui signifie qu’il est gratuit et que tout le monde peut participer à son développement, car son code est rendu public.

Observation des Développeurs

Enfin, j’ai suivi le travail des développeurs durant les trois derniers jours. J’ai pu observer Erwan Fournit déboguer (trouver et corriger les erreurs du code) et ajouter de nouvelles fonctionnalités dans une application web appelée DeskActiv. Cette application est destinée aux bureaux d’études et est codée à l’aide de plusieurs langages de programmation (PHP, Typescript, HTML et CSS).

DeskActiv utilise le framework Angular pour gérer les requêtes entre PHP, Typescript et HTML. Dans cette application, PHP sert à créer une API (Application Programming Interface) qui permet à deux applications de communiquer entre elles, comme MySQL et Typescript.

MySQL est un système de gestion de base de données relationnelle qui utilise SQL pour ses requêtes. Typescript est un langage de programmation similaire à JavaScript mais plus orienté objet, ce qui lui permet de gérer un ensemble de valeurs (nombres, chaînes de caractères, tableaux, etc.). Le langage HTML organise les éléments de la page web et le langage CSS gère la mise en forme.

On voit que, comme pour une grande partie des logiciels, le code de DeskActiv est constitué de 2 grandes parties :

  • Le frontend (« front ») qui sert à gérer les actions faites par l’utilisateur (Typescript, HTML et CSS).
  • Le backend (« back ») qui est le moteur de l’application dont l’utilisateur n’a pas connaissance.
